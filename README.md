# sports-assets

Photo library for picklesg.com, picklejohor.com, picklekualalumpur.com, padelsingapore.com, padeljohor.com.

## Structure

```
pickle/
  sg/           — Singapore pickleball courts, venues, scenes
  my/johor/     — Johor Bahru pickleball
  my/kl/        — Kuala Lumpur pickleball
  generic/      — paddles, balls, gameplay (country-neutral)

padel/
  sg/           — Singapore padel courts, venues, scenes
  my/johor/     — Johor Bahru padel
  generic/      — rackets, balls, gameplay (country-neutral)
```

## Usage in blog posts

Reference images via raw GitHub URL:
```
https://raw.githubusercontent.com/whooshinglander/sports-assets/main/pickle/sg/kallang-courts.jpg
```

## Rules
- Pickleball photos go in `pickle/`, padel photos in `padel/`
- Country/city-specific shots must be in the correct country subfolder
- Generic gameplay/gear shots go in `generic/`
- Filename should describe the subject: `kallang-courts.jpg`, `paddle-and-ball.jpg`
- Only use photos that are free to use (Unsplash licence, CC0, or venue-provided press photos)
- Do NOT mix SG and MY venue photos — country specificity matters
